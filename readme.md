CRUD Python com SQLAlchemy

  Um projeto de exemplo que demonstra operações CRUD (Create, Read, Update, Delete) em Python
  utilizando SQLAlchemy como ORM e SQLite como banco de dados.

Tecnologias Utilizadas

  Python 3.12+
  SQLAlchemy 2.0+ - ORM para mapeamento objeto-relacional
  SQLite - Banco de dados embutido
  Poetry - Gerenciador de dependências e ambiente virtual

Para Testar 

  python script.py
